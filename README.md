## Mur Translator API

**Mur Translator API** — это простой текстовый API, который переводит обычный текст на **мурский язык** с использованием словаря и правила добавления суффикса **"ость"**.

API может использоваться в ботах, приложениях, сайтах или скриптах.

---

# Как работает Mur API

Работа API состоит из нескольких этапов.

## 1. Получение текста

Клиент (например Telegram-бот, сайт или программа) отправляет текст для перевода.

Пример запроса:

```
я иду в школу
```

---

## 2. Разделение текста

API разбивает текст на слова.

Пример:

```
["я", "иду", "в", "школу"]
```

---

## 3. Проверка словаря

Каждое слово проверяется в словаре.

Словарь:

```
dictionary = {
    "я": "ч",
    "друг": "друн",
    "течток": "течтокерность",
    "школа": "школьность",
    "ты": "тобость",
    "мы": "намость"
}
```

Если слово есть в словаре — используется готовый перевод.

---

## 4. Применение правила

Если слова **нет в словаре**, API добавляет суффикс:

```
ость
```

Пример:

```
иду → идуость
в → вость
```

---

## 5. Сборка текста

Все переведённые слова объединяются обратно в предложение.

Результат:

```
ч идуость вость школуость
```

---

# Пример работы API

### Запрос

```
POST /translate
```

Тело запроса:

```json
{
  "text": "я иду в школу"
}
```

---

### Ответ

```json
{
  "translated": "ч идуость вость школуость"
}
```

---

# Алгоритм перевода

Алгоритм работы API:

1. Получить текст
2. Разделить текст на слова
3. Для каждого слова:

   * привести к нижнему регистру
   * проверить в словаре
   * если есть → заменить
   * если нет → добавить `"ость"`
4. сохранить регистр первой буквы
5. собрать предложение
6. вернуть результат

---

# Пример функции API

```python
def mur_translate(text):
    words = text.split()
    result = []

    for w in words:
        lw = w.lower()

        if lw in dictionary:
            translated = dictionary[lw]
        else:
            translated = lw + "ость"

        if w[0].isupper():
            translated = translated.capitalize()

        result.append(translated)

    return " ".join(result)
```
